<h2>About</h2>
<p><b>Wavetable Synth</b> is a synthesizer plugin built with <b>C++</b> and <b>JUCE</b> that lets you generate sound using an <b>interactive GUI</b>. Use it with DAWs like Ableton Live or Reaper to play in real time or to make music!</p>
<h2>Features</h2>
<ul>
  <li><b>Waveform interpolation</b>: Adjust the shape slider to blend between sine and square waves.</li>
  <li><b>Real-time waveform display</b>: See the waveform update as you make adjustments.</li>
  <li><b>Audio controls</b>: Adjust fade-in, fade-out, volume and pan using their respective sliders.</li>
  <li><b>Plugin integration</b>: Install as a VST3 plugin and use in Ableton Live, Reaper or other DAWs.</li>
</ul>
